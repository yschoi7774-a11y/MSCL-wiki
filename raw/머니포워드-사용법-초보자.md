## 서비스 개요

- **정식 명칭**: 머니포워드 클라우드 회계
- **요금**: 초기비용 없음, 월액 3,980엔~(플랜별 상이, 연간 계약 기준)
- **특징**: 클라우드 기반 — 인터넷 환경이면 어디서든 접속 가능

## 시작 전 준비물

- 전기 결산서류
- 계정과목 체계
- 인터넷뱅킹·카드사 로그인 정보
- 기존 회계소프트웨어 데이터 (이전 사용 시)

## 초기 설정 4단계

```
① 가이드 확인 (이 문서)
② 초기 설정 ① — 사업자 설정 (법인명·결산월·소비세 설정 등)
③ 초기 설정 ② — 분개 입력 설정 (금융기관 연동)
④ 분개 입력 시작 → 입력 내용 확인
```

## 핵심 기능

### 1. 금융기관 자동 연동
- 연동 가능 기관: 전국 2,300사 이상 (은행·신용카드·전자머니 등)
- 연동 방식 3종:
  - **API 방식** — 가장 안정적
  - **ID/비밀번호 방식**
  - **전자증명서 방식**
- 연동 후 거래 명세가 홈 화면에 자동 표시됨

### 2. 자동 분개 (AI 학습)
- 금융기관 연동 명세 → AI가 계정과목 자동 추천
- 거래 패턴을 학습하여 다음 번부터 정확도 향상
- **자동 분개 규칙** 설정 시 반복 거래 완전 자동화 가능

### 3. 수동 분개 4가지 방법

| 방법 | 설명 | 추천 대상 |
|------|------|---------|
| 전표 입력 | 전표 양식으로 1건씩 입력 | **초보자 최우선 추천** |
| 분개장 입력 | 스프레드시트처럼 고속 입력, 키보드 중심 | 숙련자 |
| 파일 임포트 | CSV·Excel 일괄 입력 | 대량 데이터 |
| 장부에서 직접 입력 | 총계정원장 등 장부에서 직접 | 확인·수정 시 |

### 4. 자동 장표 생성
- 대차대조표·손익계산서 — 원클릭 생성
- 결산서, 총계정원장 등 다양한 장부 대응
- 세무사와 실시간 데이터 공유 가능

## 일상 업무 흐름

```
은행·카드 연동
    ↓
거래 명세 자동 수집
    ↓
AI 자동 분개 제안
    ↓
확인·수정 (수동 보완)
    ↓
장표 자동 생성
    ↓
세무사 공유 / 결산
```

---

## 초보자 팁

- **처음에는 전표 입력**부터 시작 — 입력 방법이 불확실할 때 가장 안전
- 금융기관 연동을 먼저 설정하면 일상 입력의 80% 이상 자동화됨
- AI가 학습하므로 초반엔 분개 제안을 하나씩 확인하고 수정하는 것이 중요
- 샘플 데이터로 먼저 연습 가능 (실제 데이터에 영향 없음)
