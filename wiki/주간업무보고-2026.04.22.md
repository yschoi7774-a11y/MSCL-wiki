# 주간업무보고 2026.04.22 (통합)

> [!tip] 서포트팀 이사 관점 핵심 takeaway
> 이번 주 가장 주의할 신호 두 가지: **① [[베베스완]] AS 급증** (1~3월 월평균 10건 → 4월 상반기만 15건, 초기불량 포함) — 판매량 증가와 비례하지만 품질 관리 임계점에 도달할 가능성 있음. **② 쿠션 제조단가 인상** (kg당 400원 → 쿠션 600원+, 추가 인상 여지) — 원가 상승이 가격 정책에 영향을 주기 전에 마진 시뮬레이션이 필요하다. 영업이사 채용·SG인증·창고이전 등 굵직한 과제가 동시 진행 중이므로 우선순위 명확화가 요구된다.

|----------|------|------|

| 조직 구조·업무 효율성 개선 | 브리핑 준비 중 (전직원 대상) | 업무분장 재정비, 직무기술서, 위임전결, SOP, 성과측정 5개 과제 |

### 금주 완료·진행

- **호주 sell out 보고**
- **4월 급여데이터** 작성 → 노무사 송부 완료
- **사무실 인테리어** 견적 (계속)
- **Money Forward** 회계시스템 내부 교육 (계속) → [[머니포워드-회계시스템]]
- **SG 인증** 제품 사전검사 — 인증협회 위탁기관에 검사 의뢰
- **신규 인턴 야마모토 히카리** 근무 시작
  - 템플대학 아시아학과 4학년
  - 일본어·한국어·영어·중국어(초급) 가능
- **블랭킷 ASTM·CPSIA 검사** — SGS Japan 협의 중
- **외부업체 할인율 기준** 신설
- **[[출장관리규정]]** 신설 완료 (2026.04.17 서명)
- **[[영업이사-채용]]** — 비즈리치와 계약 진행 중, 골든위크 전 공고 예정
- **브릴리아 아리아케 계약 연장** — 임차료 30만엔 + 관리비 2만엔 = **월 32만엔** (최초 제안 32만엔 → 2만엔 인하 협상 성공)

### 차주 중점

- SG 인증 공장검사 일정 협의 (2박 3일 예정)
- 베베스완 측 안전·품질 관련 매뉴얼 요청 중 → 완성 후 5~6월 공장 실사 목표

### 주의 신호 ⚠️

> **베베스완 AS 접수 급증**
> - 1~3월 누계: 31건 (월평균 ~10건)
> - 4월 1~20일: **15건** (초기불량 10건 / 사용 중 파손 5건)
> - 판매량 증가 비례이나, 모니터링 강화 필요

## 물류팀 (정유엽)

### KPI 현황

| 항목 | 연간 목표 | 비고 |
|------|---------|------|
| 오배송률 | 0.2% (25년 0.18%) | 수치 미기재 |
| 재고 관리 정확도 | 98% (25년 97.9%) | 수치 미기재 |

### 주요 진행 사항

**베베스완 재고·발주**
- 38차 2,000개 → 5/6 출고 예정
- 39차 2,000개 → 5/11 출고 계획 중
- 현 재고: 4/6 약 2,500개 → 4/20 **약 1,388개** (감소 추세)
- 5/1 YKK 부자재 일본 선적 예정 — 스케줄 단축 요청 중

**창고 이전 준비 비용**

| 항목 | 금액 |
|------|------|
| 팔레트 50장 | 50,000엔 |
| 팔레트렉 24개 | 348,000엔 |
| 운반비 | 25,000엔 |
| **소계** | **423,000엔** |
| 주말 물품 운반(1일) | 11~13만엔 |
| 보안카메라 설치 | 별도 |

**인력**
- 이나가키 (1993년생) 채용 확정, **5/7 출근 예정**
- 동유모 아르바이트 공고 재게재 (시급 1,400엔 → 1,500엔 인상)

**박스히어로 (재고관리 프로그램)**
- 전체 SKU 등록 완료
- 제품 바코드·이미지 입력 예정
- 모바일로 SKU·바코드·이미지 실시간 확인 가능

**원가 변동 ⚠️**
- **쿠션 제조단가 인상**: 휴비스 솜 kg당 400원 → 쿠션 600원 인상
- 쿠션 외피 모두 인상, 석유수급 불균형 원인, **추가 인상 여지 있음**

**쎄택 장기재고** — 3~4파레트 폐기 예정, 실비용 확인 후 정용대표와 협의

### 차주 중점

- ASTM 라벨링 패스 여부 확인
- 베베스완 38차 발주 및 출고일 협의
