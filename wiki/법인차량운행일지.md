# 법인차량 운행일지

> [!tip] 서포트팀 이사 관점 핵심 takeaway
> 운행일지는 **세무·법인세 관점에서 필수 증빙**이다. 차량 운행 목적이 '업무용'임을 입증하는 유일한 서류이므로, 미기재·허위 기재 시 개인 사용으로 간주되어 급여 과세 대상이 될 수 있다. 매 운행 후 즉시 기록하는 습관이 핵심이며, 월말 합산 후 경리팀에 제출하는 플로우를 명문화해야 한다.

---

## 양식 개요

| 항목 | 내용 |
|------|------|
| 양식명 | 법인차량 운행일지 (運行日誌) |
| 기록 단위 | 운행 1건마다 1행 기입 |
| 관리 주기 | 월별 시트 (연도별 파일) |
| 파일 형식 | Excel (.xlsx) |

---

## 기입 항목

| 컬럼 | 일본어 | 설명 |
|------|--------|------|
| 운행일 | 運行日 | 해당 날짜 |
| 차량번호 | 車両番号 | 법인 차량 번호판 |
| 운전자 | 運転者 | 운전자 성명 |
| 목적지 | 目的地 | 방문처·행선지 |
| 용도 | 用途 | 운행 목적 |
| 차량상태 확인 | 車両状態の確認 | 이상 없음 시 V 체크 |
| 비고 | 備考 | 특이사항 |

**용도 예시**: 거래처 방문 / 납품 / 미팅 / 회의 참가 등

**차량상태 확인 항목**: 외관 상태(흠집·파손), 타이어 공기압·마모 등

---

## 운용 절차

```
운행 전
  └─ 차량상태 확인 (V 체크)

운행 후 즉시
  └─ 운행일·차량번호·운전자·목적지·용도 기입

월말
  └─ 해당 월 시트 완성 → 경리팀 제출 또는 공유드라이브 보관
```

---

## 세무·법적 포인트

- 법인 차량을 업무용으로 인정받으려면 운행일지 **보관 의무** (세무조사 시 제출 요구 대상)
- 운행 목적이 불명확하거나 일지 미비 시 → 개인 사용으로 간주 → **현물급여로 과세**
- 보관 기간: 최소 **7년** (법인세법 기준)
