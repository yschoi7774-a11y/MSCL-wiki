# 품질·안전 관리 매뉴얼 (SG인증)

> [!tip] 서포트팀 이사 관점 핵심 takeaway
> 이 매뉴얼은 **베베스완의 SG인증 유지를 위한 핵심 운영 문서**다. 공장검사 시 담당자가 셀프체크시트를 작성할 때 이 매뉴얼의 각 항목을 근거로 삼아야 한다. 실무적으로 가장 중요한 포인트는 ① 불만 접수 24시간 이내 대응, ② 설계 변경 시 반드시 재시험, ③ SG검사원 자격 연 1회 갱신 — 이 세 가지가 누락되면 인증 유지에 직결된다.

## 핵심 내용 요약

### 1. 조직 관리
- 품질관리 책임자(QM 책임자) 지정 및 권한·책임 문서화 필수
- SG인증 업무 담당자 지정, 최신 정보 유지
- 인사이동·조직 변경 시 즉시 조직도 개정 (개정번호·날짜·승인자 명기)

### 2. 문서 관리
- 적용 대상: 사내규정, 품질 기록, 외부 문서(SG기준·법규)
- SG기준 개정 정보 연 1회 이상 확인
- 폐지 문서: '폐지' 스탬프 후 소정 기간 보관 → 폐기

### 3. 불만 처리
- **접수 후 24시간 이내** 초기 대응 실시
- 안전 관련 중대 불만 → 접수 즉시 관리책임자 보고
- 원인 규명: 피시본 다이어그램 또는 왜왜분석 사용
- 재발 방지책 수립 후 유효성 확인될 때까지 모니터링

### 4. 설계 관리
- 설계 사양이 SG기준 충족 여부를 '설계 적합성 확인표'에 기록
- **설계 변경 시 안전성 영향 평가 → 필요한 경우 재시험 필수**
- 설계 기록 보관 기간: 제품 단종 후 최소 **5년**

### 5. 자재 관리
- 입고 자재: 수입 검사 전 '미검사' 표시 → 검사 후 합격/불합격 라벨 부착
- 불합격 로트: 격리 보관 → 품질관리부 보고 → 책임자가 처치 결정 (반품·폐기·특별채용)
- 공급업체에 시정 요구 및 재발 방지 확인

### 6. 공정 관리
- 제품별 QC 관리 공정도 작성 (공정명·설비·관리특성·기준값·측정방법·기록방법·담당자)
- 설계 변경·공정 변경 시마다 QC 공정도 개정, 제조 현장에 최신판 게시
- 특수 공정(용접·열처리·도장·성형 등): 후공정 검사로 확인 불가 → 별도 관리

### 7·8. 설비·계측기기 관리
- 설비 대장 등록 및 보전 이력 관리
- 계측기기: 교정 기한 관리, 기한 초과 방지
- 교정 불합격 기기: '사용 금지' 표시 후 즉시 사용 중지 + 과거 측정 결과 유효성 평가

### 9. SG 적합성 검사
- 출하 전 전 제품에 적합성 검사 실시
- **SG 검사원** 자격 필요 (SG기준 이해도 시험 합격 + OJT 실습 수료)
- **검사원 자격 연 1회 갱신 확인**
- 치명적 결함: c=0 (1개라도 불합격이면 로트 전체 불합격)

### 10. 부적합 관리
- 불합격 로트: '불합격' 태그 + 격리 → '부적합품 관리표' 발행 → 책임자 결정
- **특별채용(특채) 불가 조건**: 안전 특성(안전성에 직접 영향)의 부적합 + SG기준 저촉 항목

## 공장검사 셀프체크시트

공장검사 시 심사 담당자에게 제출하는 공식 양식. 10개 챕터별 체크 항목 총 **48문항**.

### 판정 기준

| 판정 | 의미 | 기재 방법 |
|------|------|---------|
| Y (Yes) | 규정 존재·실시 중 | 사내규정 문서명·규정번호 기재 |
| N (No) | 규정 미정비 | '없음' 기재 + 정비 계획 검토 |
| NA (Not Applicable) | 해당 없는 항목 | 이유 명기 |

### 챕터별 체크 항목 수

| 챕터 | 항목 수 | 주요 확인 내용 |
|------|--------|--------------|
| 1. 조직 | 2 | 사내규정명, 조직도 유무 |
| 2. 문서 | 5 | 적용 범위, 배포·회수, 제정·개폐, 보관기간, SG기준 외부문서 관리 |
| 3. 불만 처리 | 6 | 적용 범위, 접수 방법, 처리·원인규명·재발방지 절차 |
| 4. 설계 | 4 | 설계 절차, 검증, SG기준 대비 목표 품질 |
| 5. 자재 | 4 | 수입 검사 범위·방법, 부적합 로트 처리 |
| 6. 공정 | 5 | QC 공정도, 작업표준서, 특수 공정·인원 인정 기준 |
| 7. 설비 | 3 | 적용 범위, 일상·정기 점검 방법 |
| 8. 계측기기 | 3 | 적용 범위, 일상·정기 교정 방법 |
| 9. SG 적합성 검사 | 11 | 로트 구성, 샘플링, 합부 판정, 검사 절차·담당자·기록·승인자 |
| 10. 부적합 관리 | 7 | 불합격 로트·불량품 조치, 특채 범위·절차·승인자 |

> [!note]- 작성 시 주의사항
> - Y 판정 시: 사내규정 명칭 또는 문서 번호를 반드시 기재 (예: "문서 규정", "부적합품 관리 규정")
> - N 판정 시: "없음" 기재 후 정비 계획 수립 필요
> - 사전 서면 심사 시 본 체크시트를 심사 담당자에게 제출
