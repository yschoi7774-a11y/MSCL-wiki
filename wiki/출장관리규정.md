# 출장 관리 규정

> [!tip] 서포트팀 이사 관점 핵심 takeaway
> 임원 기준 여비 한도: **숙박 7,000엔/박, 식비 5,000엔/일, 수당 6,000엔(숙박)/4,000엔(당일)**. 7,000엔 초과 숙박은 팀장 사전 승인 후 예약해야 환급 가능하므로, 출장 전 계획서 단계에서 반드시 처리할 것. 사전 승인 없는 출장은 여비 지급이 제한됨 — 직원 출장 승인 시 이 점을 명확히 안내해야 한다.

---

## 개요

- **시행일**: 2026년 4월 17일
- **서명**: 대표이사 박정우 (전자서명 완료)
- **목적**: 출장 절차·승인 권한·여비 기준 통일, 불필요한 비용 방지

---

## 출장 절차 (3단계)

```
① 출장계획서 제출 → 사전 승인
② 출장 수행
③ 출장 보고서 제출 → 5영업일 이내 결재 완료
```

> ⚠️ 사전 승인 없이 진행된 출장은 여비 지급 제한

**결재 경로**: 담당자(기안) → 팀장(검토) → 임원(승인)

---

## 복귀 원칙

| 구분 | 원칙 |
|------|------|
| 당일 복귀 | 원칙 |
| 익일 복귀 | 현지 부가 업무로 근무시간 보충 필요 |
| 개인 사정 체류 | 숙박비·식비·수당 미지급, 추가 항공권 비용 본인 부담 |

**업무 종료 시점**
- 국내: 현지에서 모든 업무를 마친 때 (본사 귀환 시 사후 정리 완료 시점)
- 국외: 귀국 후 공항 입국 수속 완료 시점

---

## 여비 기준표

### 교통비 (실비)
| 수단 | 기준 |
|------|------|
| 항공 | 이코노미 클래스 |
| 철도 | 일반석 |
| 택시 | 대중교통 이용 곤란한 경우에 한함 |

### 숙박비
- **상한**: 1박 **7,000엔** (실비)
- 초과 시: 출장계획서 단계에서 **팀장 사전 승인** 후 예약·결제

### 식비 (실비, 영수증 필수)
| 직급 | 한도 |
|------|------|
| 임원 | **5,000엔**/일 |
| 직원 | **3,000엔**/일 |

- 한도 초과 시 영수증 + 사유서 제출

### 출장수당 (일수 기준, 개인 체류기간 제외)

**국내 출장**
| 구분 | 당일치기 | 숙박 포함 |
|------|---------|---------|
| 임원 | 4,000엔 | 6,000엔 |
| 관리직 | 3,000엔 | 4,000엔 |
| 일반사원 | 2,000엔 | 3,000엔 |

**국외 출장** (국내 수당 기준에 배율 적용)
| 지역 | 배율 |
|------|------|
| 북미·남미·유럽 | 1.5배 |
| 아시아·아프리카 | 1.2배 |

- 특수 사정(치안·물가 등) 시: 대표이사 사전 승인으로 별도 금액 적용 가능

- **예외**: 부득이 시 개인카드·현금 결제 후 영수증 첨부 정산
