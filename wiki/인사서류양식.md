# 인사 서류 양식

> [!tip] 서포트팀 이사 관점 핵심 takeaway
> 서면 양식과 디지털 시스템([[인사정보시스템-연차관리]])이 **이중으로 운용**된다. 종이 양식은 날인 기록이라는 법적 근거, 시스템은 이력 추적 역할 — 양식 제출 시 시스템 동시 반영을 의무화하지 않으면 수치가 엇갈린다. 직원 입사 시 전체 양식 세트를 안내하고, 퇴직·주소변경 등 이벤트 발생 시 해당 양식을 즉시 제출받는 프로세스를 명문화할 것.

## 유급휴가 신청서

**기입 항목**: 신청일 / 소속 / 성명 / 휴가 기간 / 구분(전일·반일) / 취득 일수 / 긴급연락처 / 비고 / 승인란

**처리 흐름**:
1. 직원이 양식 작성 → 담당자 제출
2. 승인자 날인
3. [[인사정보시스템-연차관리]]에 사용 내역 입력 (누락 주의)

> [!note]- 개선 과제
> - 서면 신청 → 시스템 입력 동시화 프로세스 명문화
> - 반일 휴가(오전/오후) 구분 시스템 반영 여부 확인
> - 전자결재 전환 검토

---

## 퇴직서

**기입 항목**: 성명 / 퇴직 연월일 / 퇴직 사유 / 소속 / 직위 / 작성일 / 자필 서명

**수리 후 처리**:
1. 원본 인사 파일 보관
2. 사회보험 상실 신고 — 건강보험·후생연금 (5일 이내)
3. 고용보험 상실 신고 (10일 이내)
4. 원천징수표 발행 / 퇴직금 정산 (해당 시)

---

## 주소변경 신청서

**기입 항목**: 성명 / 소속·직위 / 구 주소 / 신 주소 / 변경일 / 서명

**처리 흐름**:
1. 신청서 + 주민표 사본 제출
2. 인사 정보 갱신
3. **통근수당 재산정** (거리 변동 시 지급액 변경)
4. 사회보험 주소 변경 신고 (해당 시)
