# 머니포워드 클라우드 회계 (Money Forward)

> [!tip] 서포트팀 이사 관점 핵심 takeaway
> 2026년 하반기 병행 도입 예정. 가장 중요한 것은 **금융기관 연동을 먼저 설정**하는 것 — 이것만 해도 일상 입력의 80%가 자동화된다. 병행 기간 동안 기존 시스템과의 이중 입력 부담이 생기므로, 도입 전에 ① 전환 일정, ② 담당자, ③ 계정과목 기준을 통일해두지 않으면 혼선이 커진다. 서포트팀이 내부 교육을 주도하고 있으므로 전 직원 교육 완료가 선행 과제다.

## 시작 전 준비물

- 전기 결산서류
- 계정과목 체계
- 인터넷뱅킹·카드사 로그인 정보

## 초기 설정 4단계

```
① 가이드 확인
② 사업자 설정 — 법인명·결산월·소비세 설정
③ 분개 입력 설정 — 금융기관 연동
④ 분개 입력 시작 → 입력 내용 확인
```

## 핵심 기능

### 1. 금융기관 자동 연동
- 연동 가능 기관: 전국 **2,300사 이상** (은행·신용카드·전자머니 등)
- 연동 방식 3종: API 방식(가장 안정적) / ID·비밀번호 방식 / 전자증명서 방식
- 연동 후 거래 명세가 홈 화면에 자동 표시됨

### 2. 자동 분개 (AI 학습)
- 연동 명세 → AI가 계정과목 자동 추천
- 거래 패턴 학습으로 시간이 지날수록 정확도 향상
- **자동 분개 규칙** 설정 시 반복 거래 완전 자동화 가능

### 3. 수동 분개 입력 방법

| 방법 | 설명 | 추천 대상 |
|------|------|---------|
| **전표 입력** | 전표 양식으로 1건씩 입력 | **초보자 — 여기서 시작** |
| 분개장 입력 | 스프레드시트처럼 고속 입력 | 숙련자 |
| 파일 임포트 | CSV·Excel 일괄 입력 | 대량 데이터 |
| 장부에서 직접 입력 | 총계정원장 등 장부에서 확인·수정 | 수정 작업 시 |

### 4. 자동 장표 생성
- 대차대조표·손익계산서 — 원클릭 생성
- 결산서, 총계정원장 등 다양한 장부 대응
- 세무사와 실시간 데이터 공유 가능

## 일상 업무 흐름

```
은행·카드 연동
    ↓
거래 명세 자동 수집
    ↓
AI 자동 분개 제안
    ↓
확인·수정 (수동 보완)
    ↓
장표 자동 생성
    ↓
세무사 공유 / 결산
```
